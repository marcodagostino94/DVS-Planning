DVS PLANNING BUILD 13.3 — PASSAGGI OBBLIGATORI

1. In Supabase SQL Editor eseguire:
   database/010_backup_agent_status_build_13_3.sql

2. Generare il nuovo PKG dal file DVS_Backup_Agent_0.4.0.zip:
   chmod +x tools/build_pkg_macos.sh
   ./tools/build_pkg_macos.sh

3. Installare il nuovo PKG 0.4.0 sopra la versione 0.3.1.
   Configurazione, stato, log e cartella backup vengono conservati.

4. Copiare il nuovo PKG nella cartella downloads del Planning e rinominarlo esattamente:
   DVS_Backup_Agent.pkg
   Sostituire il PKG 0.3.1 già presente.

5. Solo dopo eseguire commit e push della Build 13.3 su GitHub.

La pagina Backup mostra stato, ultimo backup, utente, computer e versione.
Il pulsante Revoca autorizzazione blocca i backup successivi senza eliminare i file già creati.
